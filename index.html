Library support in your IDE of choice. Android Studio, Eclipse, Idea, NetBeans...
Gradle build file
After the android block, there is a block that shows the libraries used for the app.
The dependencies section shows values for the compile, testCompile, and androidTestCompile configurations.
The project build.gradle file should have contents similar to 
dependencies {
	// ...
    compile ' com.flyvemdm:inventory:x.x.x' (1)
}
